# NWPU-VHR-10_2_VOC

reference blog: https://blog.csdn.net/practical_sharp/article/details/114764624
First, download the NWPU-VHR-10 dataset
Second, use the 'NWPU-VHR-10_2_voc.py' file to transfer format
Third, split the dataset into train.txt and val.txt using the 'split.py' file and put them in the directory 'ImageSets/Main'
What's more, maybe you need the 'voc_classes.txt' file
